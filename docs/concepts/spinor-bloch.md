# Spinors and Bloch sphere

Spinors and Bloch vectors are two views of single-qubit state information.

---

## Spinor form

```text
|psi> = alpha|0> + beta|1>
|alpha|^2 + |beta|^2 = 1
```

---

## Bloch mapping

A normalized spinor maps to a point on the Bloch sphere:

```text
n = (sin(theta)cos(phi), sin(theta)sin(phi), cos(theta))
```

Global phase does not change the Bloch point.
